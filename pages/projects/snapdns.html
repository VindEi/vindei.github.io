SnapDNS

A Cross-platform DNS manager. Consists of a Clean Flutter UI And a high-performance C# .NET 10 service(Native Code on Android)

Download View Source
Tap image to switch theme

Split-Process Architecture

The unprivileged user interface (Flutter) runs in user-space, communicating via secure IPC (Named Pipes & Sockets) with a native background system service (C# .NET 10).

Cross-Platform Client

Unified management client designed for Windows, macOS, Linux, and Android, giving you the same control on all your devices.

Encrypted DNS (DoH/DoT)

Configure, manage, and enforce secure DNS-over-HTTPS and DNS-over-TLS protocols easily across all active network adapters.

Preset & Custom Profiles

Swap between fast pre-configured servers (Cloudflare, Google, Quad9) or construct your own custom DNS profiles with ease.

Integrated Diagnostics

Features built-in tools like Latency Ping Check (to find the fastest DNS), Flush DNS Cache directly inside the app.

Open & Transparent

Zero telemetry, fully open-source (GPLv3), and running entirely client-side. Inspect the code and manage your connections with complete peace of mind.
